# Pururi-Quest3

- **ベスラン学校占拠事件**（ベスランがっこうせんきょじけん）は、2004年9月1日から9月3日にかけてロシアの北オセチア共和国ベスラン市のベスラン第一中等学校で、**チェチェン共和国独立派**を中心とする多国籍の武装集団（約30名）によって起こされた占拠事件。

- 9月1日に実行された占拠により、7歳から18歳の少年少女とその保護者、1181人が人質となった。
3日間の膠着状態ののち、9月3日に犯人グループと治安部隊との間で銃撃戦が行われ、治安部隊が建物を制圧し事件は終了したものの、386人以上が死亡（うち186人が子供）、負傷者700人以上という犠牲を出す大惨事となった。

- 首謀者はチェチェン人のシャミル・バサエフ。
